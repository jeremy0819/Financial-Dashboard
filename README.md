## 📊 專案簡介

單一 `index.html` 靜態網頁，輸入股票代號即抓取**真實財報數據（FinMind 日收盤）**並產出**五層次深度分析報告**。所有數字皆標示資料日期（非盤中即時），靜態策展觀點與即時數據明確分流。
資料來源整合 [My TW Coverage](https://github.com/Timeverse/My-TW-Coverage)（1,735 家台股研究資料庫）的研究框架。

### 核心理念：建築思維 × 財務分析

| 建築概念 | 財務對應 |
|---|---|
| 建案預估總銷 | 損益表營收 |
| 結構強度 / 建融壓力 | 負債比 / DSCR |
| 案場周轉金 | 自由現金流 FCF |
| 開發案 IRR | 股票 ROE / 報酬率 |

## ✨ 五層次深度分析框架

```
Layer 1 — 財務健檢    即時數據訊號（均線/估值/營收/毛利/籌碼）+ 技術均線圖 + 月營收圖 + 籌碼面（三大法人/融資券）+ 真實自由現金流
Layer 2 — 估值分析    即時 P/E、P/B、現金殖利率、DCF 折現 + 股利政策（現金股利/近12月殖利率）
Layer 3 — 成長潛力    市場規模、產業增長率、AI優勢、5/10年展望
Layer 4 — 多空辯論    多頭 vs 空頭分析師，數據支持的對立論點
Layer 5 — 投資決策    BUY / HOLD / AVOID，催化劑與風險清單
```

## 🔀 雙引擎：即時數字 + 質化分析

| 引擎 | 來源 | 說明 |
|---|---|---|
| **真實數字** | FinMind API（日收盤，非盤中即時；台股當日、美股約延遲 1 交易日） | 股價/漲跌、P/E、P/B、殖利率、P/E 一年百分位、月營收年增、財報利潤率、EPS、**三大法人買賣超、融資融券餘額、真實自由現金流、現金股利/殖利率、公司名稱與產業別**（台股完整；美股提供股價與均線） |
| **質化分析** | 內建 DB / Claude AI | 6 支精選個股有策展分析；填入 Anthropic API Key 後，由 Claude 以即時數字為依據生成五層分析 |

- **不填金鑰（預設）**：即時數字 + 內建質化分析（DB 或自動生成）。
- **填入 `sk-ant-...` 金鑰**：即時數字 + Claude 即時生成的質化分析（`temperature:0`，結果快取確保一致）。

> ⚠️ 金鑰僅存在瀏覽器記憶體、直接呼叫 Anthropic（靜態網頁無後端代理），僅適合個人 Demo 使用。

## 📁 個人資產分析（核心-衛星雙軌制）

切換側欄「📁 個人資產」即進入投資組合分析，依「核心-衛星雙軌制」框架管理持股並自動檢核配置紀律：

- **建立組合**：可「載入雙軌制範本」（內建觀察清單與軌道分類，股數留空自填）或手動新增；每檔設定代號、股數、成本、軌道（核心／衛星／防禦）。
- **即時估值**：一鍵「更新報價並分析」，由 FinMind 抓每檔現價與產業，計算市值、損益、權重。
- **三軌配置**：核心 70–80% ／ 衛星 20–30% ／ 防禦現金 >15%，圓餅圖與占比一目了然。
- **七條紀律檢核**：核心 70–80%、衛星 20–30%、防禦 >15%、單一個股 <15%、單一產業 <35%、衛星單檔 <5%、衛星總和 <30%，逐條 ✅/⚠️。
- **配置健診（免金鑰）**：依七條紀律自動產生具體建議（核心/衛星/防禦失衡、單一個股與產業集中等），**無需 API Key**；另可選填 API Key 由 Claude 給出更深入質化建議。

> 🔒 **隱私**：所有持股資料僅存在你的瀏覽器 `localStorage`，不會上傳、也不會寫入此公開儲存庫。

## 🔧 技術架構

- **前端**：HTML5 + Tailwind CSS (CDN) + Vanilla JavaScript（單一檔案，無框架、無建置）
- **即時數據**：[FinMind API](https://finmindtrade.com/)（CORS 開放、免金鑰）— 價格／PER／月營收／財報／三大法人／融資融券／現金流量／股利／公司資訊
- **圖表**：Chart.js 4（股價 × 月營收組合圖）
- **AI（選填）**：Anthropic Claude Messages API
- **部署**：GitHub Pages（純靜態）
