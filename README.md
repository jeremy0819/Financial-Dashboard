## 📊 專案簡介

單一 `index.html` 靜態網頁，輸入股票代號即抓取**真實財報數據（FinMind 日收盤）**並產出**六層次深度分析報告**。  
所有數字皆標示資料日期（非盤中即時），靜態策展觀點與即時數據明確分流。  
資料來源整合 [My TW Coverage](https://github.com/Timeverse/My-TW-Coverage)（1,735 家台股研究資料庫）的研究框架。

### 核心理念：建築思維 × 財務分析

| 建築概念 | 財務對應 |
|---|---|
| 建案預估總銷 | 損益表營收 |
| 結構強度 / 建融壓力 | 負債比 / DSCR |
| 案場周轉金 | 自由現金流 FCF |
| 開發案 IRR | 股票 ROE / 報酬率 |
| 最佳/最差開發情境 | 樂觀/悲觀情境分析 |

## ✨ 六層次深度分析框架

```
Layer 1 — 財務健檢    即時數據訊號 + 六維雷達圖 + 技術均線圖 + 月營收圖 + 籌碼面 + 真實自由現金流
Layer 2 — 估值分析    即時 P/E、P/B、現金殖利率、DCF 折現 + 股利政策
Layer 3 — 成長潛力    市場規模、產業增長率、AI優勢、5/10年展望
Layer 4 — 多空辯論    多頭 vs 空頭分析師，數據支持的對立論點
Layer 5 — 投資決策    BUY / HOLD / AVOID，催化劑與風險清單
Layer 6 — 情境分析    樂觀/基準/悲觀三情境 EPS & 目標價 + 台灣升息情境模擬 [NEW v6.0]
```

## 🆕 v6.0 新功能詳解

### 🕸️ 六維評分雷達圖
自動從即時數據計算六個維度評分（0–100）：
- **財務健康**：綜合財務評分
- **成長性**：月營收年增 + 產業增長率
- **估值吸引力**：P/E 百分位（低→高分）
- **籌碼面**：三大法人近5日買賣超方向
- **技術面**：均線多空排列
- **股利**：現金殖利率

### 📐 情境分析 Tab
每檔精選個股提供三情境分析：
| 情境 | 說明 |
|---|---|
| 🌟 樂觀 | 最佳觸發條件、EPS 預估、目標價、上行幅度 |
| 📊 基準 | 中性假設下的合理預期 |
| ⚠️ 悲觀 | 最壞情境下的 EPS 與目標價 |

加上**台灣特色：升息兩碼情境模擬**，分析升息對科技股、金融股、房地產/都更概念股的差異化影響。

### 🧮 投資計算器（全新模式）
側欄切換至「🧮 投資計算器」，提供四種工具：
1. **複利試算**：一次性投入 × 年化報酬率 × 年數，含成長曲線圖
2. **定期定額試算**：月定投 × 報酬率 × 年數，投入 vs 獲益堆疊圖
3. **72 法則**：輸入報酬率即時顯示翻倍年數
4. **台灣定存試算**：存款金額 × 利率 × 月數，含稅後利息計算

## 🔀 雙引擎：即時數字 + 質化分析

| 引擎 | 來源 | 說明 |
|---|---|---|
| **真實數字** | FinMind API（日收盤，非盤中即時） | 股價/漲跌、P/E、P/B、殖利率、P/E 一年百分位、月營收年增、財報利潤率、EPS、三大法人、融資融券、自由現金流、股利、公司資訊 |
| **質化分析** | 內建 DB / Claude AI | 精選個股有策展分析；填入 Anthropic API Key 後由 Claude 生成含情境分析的六層報告 |

## 📁 個人資產分析（核心-衛星雙軌制）

切換側欄「📁 個人資產」進入投資組合分析：
- **建立組合**：載入雙軌制範本或手動新增
- **即時估值**：FinMind 抓取現價、計算市值/損益/權重
- **三軌配置**：核心 70–80% / 衛星 20–30% / 防禦現金 >15%（圓餅圖）
- **七條紀律檢核**：逐條 ✅/⚠️
- **配置健診（免金鑰）**：規則式自動產生具體建議
- **AI 深入建議**：選填 API Key，由 Claude 給出質化健診

> 🔒 所有持股資料僅存在你的瀏覽器 `localStorage`，不會上傳。

## 🔧 技術架構

- **前端**：HTML5 + Tailwind CSS (CDN) + Vanilla JavaScript（單一檔案，無框架、無建置）
- **即時數據**：[FinMind API](https://finmindtrade.com/)（CORS 開放、免金鑰）
- **圖表**：Chart.js 4（技術均線 / 月營收 / 籌碼 / 雷達圖 / 複利曲線 / 定期定額）
- **AI（選填）**：Anthropic Claude Messages API（含情境分析生成）
- **部署**：GitHub Pages（純靜態）
